Marbure is a premium WordPress theme for marble, stone tile, and flooring businesses.

== Description ==

Marbure is a premium WordPress theme purpose-built for marble, stone tile, and flooring businesses. It combines a refined visual aesthetic with a powerful feature set:

* **Elementor Ready** — All page layouts are built with Elementor Page Builder. Includes 10 custom Elementor widgets in the "Marbure" category: Hero Slider, Services Grid, Portfolio Grid, Testimonial Carousel, Stats Counter, Team Grid, Clients Marquee, Gallery Grid, FAQ Accordion, and CTA Banner.
* **Kirki Customizer** — Deep WordPress Customizer integration via the bundled Kirki framework. Control colors, fonts, hero content, section visibility, stats, and more — all with live preview.
* **Custom Post Types** — Dedicated CPTs for Services, Portfolio, Team, Testimonials, Clients, Products (stone/tile inventory), Projects (completed installations), and FAQs.
* **Swiper 11 Sliders** — Hero slider and testimonials carousel powered by Swiper with configurable autoplay, loop, and speed settings.
* **AOS Animations** — Subtle scroll-triggered animations throughout the page.
* **GLightbox** — Lightbox gallery for portfolio and product images.
* **Schema Markup** — Structured data (JSON-LD) for LocalBusiness, Product, FAQPage, and AggregateRating.
* **Performance First** — Critical CSS inlined, lazy loading, conditional script loading, and minified assets.
* **RTL Ready** — Full right-to-left language support.
* **Translation Ready** — Fully translatable with a `.pot` file included.

== Installation ==

1. In your admin panel, go to **Appearance > Themes** and click **Add New**.
2. Click **Upload Theme**, choose the `marbure.zip` file, and click **Install Now**.
3. Click **Activate** to use the theme.
4. Install and activate the required plugin: **Elementor** (free version).
5. Go to **Appearance > Customize** to configure site identity, colors, fonts, and homepage sections.
6. Create pages and assign the provided page templates (Home, About, Services, Portfolio, etc.) via the Page Attributes box in the page editor.
7. Import the sample content (optional) from the `assets/demo/` folder using the WordPress Importer.

== Frequently Asked Questions ==

= Does this theme require Elementor? =

Yes. All page templates render their content through Elementor. Without Elementor activated, pages will appear blank. The free version of Elementor is sufficient for all bundled functionality.

= Can I use this theme without the custom widgets? =

The 10 custom Elementor widgets ship with the theme and are always available when Elementor is active. They appear under the "Marbure" widget category in the Elementor editor.

= How do I set up the homepage slider? =

Assign the "Home" page template to a page, edit it with Elementor, and drag the Hero Slider widget onto the canvas. Each slide's image, heading, subheading, button text, and link are configurable in the widget settings panel.

= Is this theme compatible with WooCommerce? =

The theme includes a `marbure_product` custom post type for displaying stone/tile products with material, size, and finish details. WooCommerce is not required but is compatible if you need full cart/checkout functionality.

= How do I change the primary colors and fonts? =

Go to **Appearance > Customize**. The Marbure section contains color pickers for the primary, secondary, accent, and background colors, as well as font selectors for heading and body typefaces.

= Where can I find documentation? =

Full documentation is included in the `docs/` folder of the download package and is also available on the ThemeForest item page.
